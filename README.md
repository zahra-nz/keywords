# keywords

this project is for finding the most frequent used words in a file (pdf)
Pdf file will be uploaded (file name not affecting the input and the code is designed as any file withany name can be generated)
libraries used: 
 PyPDF2
 spacy
 collections: Counter
 nltk
 nltk.corpus: stopwords
 sys

there are also some stopwords added mannually to be more efficient in the output.
